# T-BUG OS - 产品介绍文档

> 让AI Agent拥有记忆、学习和自主能力

## 一句话介绍

**T-BUG OS** 是一个开源的AI Agent操作系统框架，让任何人都能快速搭建具有记忆能力、自主学习能力和自动化工作流的智能AI助手。

## 背景与问题

### 当前AI助手的局限

| 局限 | 描述 |
|------|------|
| ❌ 无记忆 | 每次对话都是全新开始 |
| ❌ 被动响应 | 只能问一句答一句 |
| ❌ 无成长 | 不会主动学习新知识 |
| ❌ 无人格 | 没有长期身份和偏好 |

你是否遇到过：
- AI总是重复问同样的问题
- 教过AI的东西，下次又忘了
- 想让AI自动做点什么，但它只能等人命令

### 1. 记忆系统 - 像人一样记住一切

| 记忆类型 | 存储内容 | 用途 |
|----------|----------|------|
| **情景记忆** | 每天的对话和事件 | 记得聊过什么 |
| **语义记忆** | 知识库、概念、事实 | 记住学到的知识 |
| **程序记忆** | 技能、工作流 | 记住怎么做事情 |
| **工作记忆** | 当前任务、待办事项 | 记住现在要做什么 |
| **元记忆** | 重要里程碑、核心身份 | 长期重要信息 |

**效果：** AI不会再"失忆"，而是像人一样有连续的记忆。

### 2. 学习系统 - 自动成长

```
用户提问 → 搜索知识 → 提取重点 → 整理入库 → 下次能用
```

- ✅ **被动学习**：从与你的对话中学习
- ✅ **主动学习**：自己上网搜索新知识
- ✅ **定时学习**：每天自动看新闻、学术论文
- ✅ **知识同步**：自动整理到知识库

**效果：** AI会自己变聪明，不需要你手把手教。

### 3. 技能系统 - 模块化扩展

```
技能模板 → 复制 → 修改配置 → 立即可用
```

- 🔌 **即插即用**：复制技能文件夹即可添加
- 🛠️ **灵活配置**：通过YAML配置行为
- 📦 **开箱即用**：提供天气、提醒、搜索等基础技能

**效果：** 像搭积木一样扩展AI能力. 工作流 - 自动化

。

### 4| 类型 | 触发条件 | 示例 |
|------|----------|------|
| **心跳** | 定时（30分钟） | 检查邮件、查看消息 |
| **定时任务** | Cron表达式 | 每天早8点学习新闻 |
| **事件触发** | 特定条件 | 新消息提醒、异常报警 |

**效果：** AI可以主动做事，不需要你盯着。
